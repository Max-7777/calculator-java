Calculator that takes in an arithmetic problem as user input and provides a solution


⁍ Uses shunting-yard algorithm: infix notation -> rpn notation
⁍ Solves the result: rpn notation -> solution

https://en.wikipedia.org/wiki/Shunting_yard_algorithm
https://en.wikipedia.org/wiki/Reverse_Polish_notation
